# 🚀 Portfolio Web - Miguel Salazar

Este proyecto es mi portfolio personal como desarrollador frontend, donde presento mis habilidades, experiencia, certificaciones y proyectos destacados.

Está desarrollado con tecnologías web base (**HTML, CSS y JavaScript**) siguiendo buenas prácticas de estructura, modularización y experiencia de usuario.

## 🧠 Sobre el proyecto

Este portfolio fue diseñado con un enfoque profesional, orientado a:

* Mostrar proyectos reales y experiencia práctica

## 🛠️ Tecnologías utilizadas

* **HTML5** → estructura semántica
* **CSS3** → diseño responsive + sistema de variables (custom properties)
* **JavaScript (ES Modules)** → lógica modular y escalable
* **Bootstrap 5** → layout y componentes base
* **Font Awesome** → iconografía

---

## ⚙️ Arquitectura del proyecto

El proyecto sigue una estructura modular simple pero escalable:

```
📁 css/
📁 js/
   ├── main.js
   └── modulos/
       ├── cambiarModo.js
       ├── contacto.js
       ├── descargaCv.js
       └── year.js
📁 img/
📁 certificaciones/
📁 cv/
📄 index.html
```

### 🔹 Enfoque

* Separación de responsabilidades (cada módulo hace una cosa)
* Inicialización centralizada en `main.js`
* Uso de `ES Modules` para escalabilidad futura
